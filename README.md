# Hyderabad 360: Integrated City Exploration Platform

An interactive, high-fidelity, and premium single-page web dashboard designed to capture the past, present, and future of Hyderabad. 

## Features
1. **Live Dashboard**: Highlights current weather, quick stats, traffic state, and recent regional news.
2. **Heritage & History**: Interactive timelines chronicling the Qutb Shahis, Nizams, and modern Cyberabad IT hub.
3. **Topography & Weather**: Detailed climate charts, geographic breakdown of Deccan Plateau and Musi river.
4. **Tourist Landmarks**: Visual catalog of famous spots (Charminar, Golconda) with seat/pass booking and visual QR-code pass generation.
5. **Public Transit Hub**: Routes for Hyderabad Metro, TSRTC buses, and smartcard passes.
6. **Gastronomy & Hotels**: Booking/table reservation engines for legendary Biryani cafes (Niloufer, Paradise) and luxury hotels.
7. **Traffic Pulse**: Real-time interactive canvas showing current congestion levels across key Hyderabad corridors.
8. **Cinemas & Shows**: Complete movie theater seat booking engine.
9. **Nagarik Safety Portal**: Emergency resources, SOS trigger, and secure redirected complaint filing to the Hyderabad City Police.

## Stacks & Design
* **Frontend**: Vanilla HTML5, CSS3, Modern ES6+ JavaScript.
* **Styling**: Royal Heritage theme (Teal `#0D5C75` & Nizam's Gold `#D4AF37`) with glassmorphism effects, clean dark slate backgrounds, and custom micro-animations.
* **Storage**: Persistent `localStorage` tracking bookings and safety incident records.

## Quick Start
Open `index.html` directly in any web browser, or run a simple local static server:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000`.
